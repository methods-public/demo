# mksh

## Description

Installs [mksh](https://www.mirbsd.org/mksh.htm).

* Opscode Community Site: http://community.opscode.com/cookbooks/mksh
* Source Code: http://github.com/gregf/chef-mksh

## Requirements

### Chef

Tested on chef 11

### Platforms

* Debian 6+
* Ubuntu 14.04+
* Centos 6.5+

## Recipes

### default

Installs and keeps mksh upgraded.
