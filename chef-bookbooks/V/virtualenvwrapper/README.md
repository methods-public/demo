# virtualenvwrapper-cookbook

Cookbook for provisioning a python virtualenv environment with a virtualenvwrapper

## Supported Platforms

* Centos 6.5

## Recipes

### Default

Installs python, pip, virtualenv and virtualenvwrapper for all users.
